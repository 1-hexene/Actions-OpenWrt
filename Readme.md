# 借助 GitHub Actions 的 OpenWrt 在线自动编译.

yml脚本默认使用mtwifi-cfg.config配置文件，配置文件含luci-app-dockerman。
在运行workflow时可选是否使用mtwifi-cfg和是否编译luci-app-dockerman，默认不使用mtwifi-cfg，默认编译luci-app-dockerman。
